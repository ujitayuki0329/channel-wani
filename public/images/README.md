# 画像ファイルの配置

このディレクトリに既存サイト（https://gustavewani.wixsite.com/fukayahachu）の画像を配置してください。

## 必要な画像ファイル

### ロゴとファビコン
- `logo.png` - サイトのロゴ（ワニのロゴ画像）
  - ファビコンとしても使用されます
  - 推奨サイズ: 512x512px または 256x256px

### ヒーロー背景画像
- `building-exterior.jpg` - 建物の外観画像
  - ヒーローセクションの背景として使用
  - 推奨サイズ: 1920x1080px 以上

### ギャラリー画像

## 画像のダウンロード方法

1. 既存サイト（https://gustavewani.wixsite.com/fukayahachu）にアクセス
2. 画像を右クリックして「名前を付けて画像を保存」
3. このディレクトリ（`public/images/`）に保存

または、ブラウザの開発者ツールを使用して画像のURLを取得し、ダウンロードしてください。

## 画像ファイル名の注意

- ロゴ: `logo.png` として保存してください
- 建物外観: `building-exterior.jpg` として保存してください
- その他の画像は既存のファイル名のまま使用できます
